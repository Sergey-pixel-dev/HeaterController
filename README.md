# HeaterController

Контроллер блока накала на STM32F446RET6. Управляет током до 30 А через выходное напряжение ЦАП.

## Функционал

- Управление силой тока блока накала через ЦАП (нелинейная зависимость ток/напряжение)
- Калибровка: измерение вольтметром → запись эталонных значений через Modbus → автоматический расчёт коэффициентов для каждого диапазона
- Тонкая настройка скорости выхода на режим и точности стабилизации
- Хранение настроек и логов в EEPROM
- Связь с ПК и другими устройствами по Modbus RTU (UART)

## Аппаратная часть

- MCU: STM32F446RET6
- Плата спаяна вручную на макетной плате с полной обвязкой
- Интерфейсы: UART (Modbus RTU), DAC, EEPROM (I2C/SPI)

## Стек

C, STM32F446, CMSIS (регистры) + HAL (только инициализация тактирования), Modbus RTU, EEPROM, DAC

## Особенности реализации

- Вся периферия (кроме системной частоты) настроена через прямую работу с регистрами
- Калибровочный алгоритм: кусочно-линейная аппроксимация с автоматическим вычислением коэффициентов по эталонным точкам
- Отладка через JTAG/SWD

## Сборка

Проект собирается в STM32CubeIDE.
